★ 1. ID 선택자 : ID가 기호(#)로 되있음 (#; 유일한값 (중복이 없음))
지정한 html 1개만 선택

★ 2. Class 선택자: class는 기호 (.)을 씀.
특정 그룹을 만들어 주는것, 유일한 값은 아님, 중복가능. 다른 태그들과 같은 class를 쓸 수 있음.
태그의 종류에 상관없이 사용 가능. Html 여러개 선택 가능

3. 전체 선택자 : 기호(*)로  모든 html 태그 선택, 주로 맨처음에 써서 전체 설정 먹일떄 씀.
